🚢 Proyecto 1 — Análisis de Supervivencia Titanic 1912
![Python](https://img.shields.io/badge/Python-Pandas%20·%20Matplotlib%20·%20Seaborn-blue)
![Status](https://img.shields.io/badge/Status-Completado-green)

Análisis exploratorio completo del dataset histórico del Titanic aplicando
limpieza de datos, análisis multivariable y visualización profesional.

**Hallazgos clave:**
- Solo el **38.2%** de los pasajeros sobrevivió
- Las mujeres tuvieron **74.2%** de supervivencia vs **18.9%** de los hombres
- **100%** de los niños de 2da clase sobrevivieron vs **45.2%** de 3ra clase
- La clase social fue determinante en el acceso a los botes salvavidas

**Habilidades:** Limpieza de datos · Análisis multivariable · Dashboard ejecutivo · Narrativa de datos
